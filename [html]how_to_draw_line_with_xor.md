## Intro
    context에 해당 옵션을 통해서 도형 합성을 진행할 수 있다.
------
1. syntax
    ```
        ctx.globalCompositeOperation = type;
    ```
2. 주 사용 목적 
    - 캔버스를 그리는데, 일부 ovarlay를 통해서 화면 겹치기가 필요한 경우 사용한다.
